Task 3 - Car Price Prediction
Overview:-
Machine learning project to predict car prices based on various features using regression models.

Objective:-
Develop and train ML models to accurately predict car prices using historical car data.


Key Features:-
Data preprocessing and feature engineering,
Exploratory data analysis,
Multiple regression models,
Model evaluation and comparison,
Price prediction on new data.


Technologies:-
Python, Pandas, NumPy,
Scikit-learn for ML models,
Matplotlib/Seaborn for visualization.
