Task 4 - Spam Email Detection with ML

Overview:-
Machine learning project to detect and classify spam emails using various classification algorithms.

Objective:-
Develop ML models to accurately identify spam emails and distinguish them from legitimate emails.

Dataset Features:-
Email text and metadata,
Various text features,
Binary classification (Spam/Not Spam).

Algorithms Implemented:-
Logistic Regression,
Naive Bayes,
Support Vector Machines,
Random Forest,
Neural Networks.

Key Metrics:-
Accuracy, Precision, Recall,
F1-Score,
ROC-AUC.
